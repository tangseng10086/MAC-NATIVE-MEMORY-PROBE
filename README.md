# Mac-Memory-Probe-C 🚀

一个用纯 C 语言编写的 macOS 内存监控探针，专为 SwiftBar 打造。
### 😃 成果展示
[!成果展示](./show.png)
### 🚩 编写目的
- 本人淘了一个二手mac，因为预算有限加上只为了出门方便所以只选择了8g内存的，为了防止莫名卡住，所以编写了这个swiftbar插件

### 🌟 核心亮点
- **极度硬核**：直接通过 macOS 内核 Mach API 获取数据，无中间商赚差价。
- **内存奇迹**：运行时内存占用仅约为 **1.0 MB**，比 Python/Shell 编写的同类插件轻量 10-50 倍。
- **实时预警**：支持内存超过 90% 状态栏自动变红报警(临界数值应该稍微大一些，防止mac特性)。
- **优雅交互**：完美适配 SwiftBar，支持下拉菜单查看详细 MB 数据。

### 🛠 如何编译
1. 确保你安装了 GCC/Clang。
2. 运行编译命令：
   ```bash
   gcc monitor.c -o mem.2s.sh
